# Clinical Task Management System

## Overview
This Python project simulates a Clinical Task Management System for healthcare professionals. It allows tracking of patient-related tasks such as medication administration, lab orders, imaging, and follow-up.

## Objectives
- Add, view, complete, and remove clinical tasks
- Track patient-specific tasks with priority levels
- Manage pending and completed tasks

## Features
- Add clinical tasks with patient name, type, and priority
- View all tasks with status
- Mark tasks as completed
- Remove tasks

## Technologies
- Python 3

## Healthcare Relevance
Task management is critical in hospital settings to ensure timely patient care. This system mimics part of hospital task tracking systems and electronic medical records (EMR) workflow.

## Future Improvements
- Integrate with SQLite for persistent storage
- Include timestamps for task creation and completion
- Multi-user support for nurses and doctors
- Develop web-based GUI for real-time hospital task tracking
